Excel template file for exports:

- logsheet_template.xlsx

This template is used by `/records/export.xlsx` so the exported workbook keeps the logo, logsheet layout, centered cells, and print settings.

Columns:
- Type of Emergency
- Chief Complaint
- Name of Patient
- Address of Patient
- Sex
- Date of Incident
- Time of Incident
- Place of Incident
- Driver
- Responders
- Communicator
- Remarks
